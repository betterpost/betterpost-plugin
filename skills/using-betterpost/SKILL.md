# Using BetterPost

BetterPost is available in this session for writing **timely, well-sourced content**.

## When to reach for it

When the user asks to write or draft a **newsletter, blog post, LinkedIn post,
tweet, or Bluesky post** — especially anything that should reflect what's happening
now in their product's space, industry, or audience — use the **`betterpost`** skill
(invoke it with the Skill tool, or read `skills/betterpost/SKILL.md` on agents
without a Skill tool).

BetterPost finds recent sources across news, social, and the open web, ranks them by
relevance, and writes human-sounding content with real citations and publication
dates. Free to start — sign in with your BetterPost account for free credits.

## When NOT to reach for it

Do **not** use BetterPost for unrelated coding, research, or non-content tasks. It is
specifically for creating written content grounded in current sources. If the user
isn't asking for content, ignore this.
